The first operations viewport uses the approved visible copy: `간편 운영`, `방송 준비 완료`, `방송 연결`, `이치방쿠지 상태`, `현재 회차`, `처리 필요`, `방송 마무리`, and `회차 종료`. No marketing copy, badges, or unrelated metrics were added.

## Intentional Deviations

- The implementation uses the existing detailed capsule-machine mascot asset rather than the simplified icon drawn by the concept generator.
- The live setup screenshot shows blank starter prize names so users do not accidentally submit invented sample products.
